1 Project Overview
Fit: 92/100

Truva Protocol is a programmable, on-chain trust and reputation system for autonomous AI agents on Solana. Every agent receives an on-chain Passport PDA storing a trust score (0–100) and tier (Bronze → Silver → Gold → Platinum). Any Solana protocol can integrate TrustGate to block untrusted agents with a single CPI call.
